# DPT

## Initial Setup

1. Install `NodeJs`
2. Install `pnpm`
3. Install `Docker Desktop`
4. Copy all `.env.example` files to `.env` and fill them out.
5. Run `pnpm install`
6. Start `Docker Desktop`
7. Run `pnpm db:deploy`
8. Run `pnpm dev`

## Subsequent Runs

1. Start `Docker Desktop`
2. Run `pnpm dev`
